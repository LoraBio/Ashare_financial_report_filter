# 《手把手教你读财报》实现索引

PDF 为扫描版，本索引依据目录页整理。每条规则后续应记录：书中页码、原意摘要、公式、适用范围、反例和测试。

1. 你必须学会读财报：排除企业、阅读框架、准备工作、重要提示、财报结构。
2. 资产负债表：货币资金、经营相关资产、生产相关资产、投资相关资产、负债和所有者权益、快速阅读。
3. 利润表：利润表重点、创造利润的过程、速读利润表。
4. 现金流量表：报表拆解、企业现金流肖像、速读现金流量表。
5. 综合阅读及分析：鸟瞰三大表、财务指标、使用财务数据估值。
6. 所有者权益变动表和财报附注。
7. 管理层汇报：摘要、董事会报告、重要事项。
8. 欺诈与反欺诈：常见操纵手法、被操纵的痕迹。

## 当前规则覆盖

- 货币资金与有息负债的安全边际。
- 应收账款占收入比重。
- 经营现金流与净利润匹配程度。
- 毛利率、净利率、ROE、资产负债率。

## 待读书验证

- 阈值不能脱离行业；增加行业基准接口。
- “异常”应能追溯到原报表科目和财报页码。
- 增加多年度趋势，避免用单年数据下结论。
- 把附注、审计意见、关联交易、资产减值和收入确认纳入证据链。
